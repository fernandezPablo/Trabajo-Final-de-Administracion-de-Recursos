COSAS POR HACER:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
[ ] COLOCAR UN INDEX EN EL DIRECTORIO RAIZ DEL SITIO QUE REDIRECCIONE A LOS USUARIOS A DONDE CORRESPONDA.
[ ] CREAR PARA EL CONTROLADOR DE USUARIO UNA FUNCION QUE PERMITA IDENTIFICAR A UN USUARIO MEDIANTE SU COOKIE DE SESION Y REIRIGIRLO A DONDE CORRESPONDA.
[X] CREAR UNA COOKIE DE SESION AL HACER EL LOGIN DEL USUARIO, REDIRECCIONAR A LA PAGINA QUE CORRESPONDA Y 	COMPROBAR LAS COOKIES EN CADA PAGINA.
[ ] EN LA PAGINA DE PANEL OPERADOR, DAR LA OPCION PARA EDITAR EN EL DETALLE DE UN CAMBIO EL IMPACTO, LA PRIORIDAD Y LA CATEGORIA.
[ ] CREAR EL CONTROLADOR PARA EL PANEL ADMINSITRADOR E IMPLEMENTAR LAS FUNCIONES DE INTERFAZ CON LA BASE DE DATOS PARA OBTENER LOS CAMBIOS EN LOS DISTINTOS ESTADOS Y MOSTRARLOS EN LA PAGINA PANEL ADMINISTRADOR.
[ ] CREAR EN LA CLASE GESTIONDB UNA FUNCION QUE PERMITA OBTENER EL SEGUIMIENTO PARA UN CAMBIO EN PARTICULAR.
[ ] CREAR EN EL CONTROLADOR PARA EL PANEL ADMINISTRADOR LA FUNCION DE INTERFAZ CON LA BASE DE DATOS PARA OBTENER EL SEGUIMIENTO DE UN CAMBIO  Y MOSTRARLO EN LA PAGINA HISTORIAL.
[ ] CREAR EN EL CONTROLADOR PARA EL PANEL ADMINISTRADOR LA FUNCION DE INTERFAZ CON LA BASE DE DATOS PARA OBTENER EL INFORME PARA UN MES EN PARTICULAR Y MOSTRAR EN LA PAGINA INFORME.
[X] CREAR EN LA CLASE GESTIONDB UN FUNCION QUE PERMITA ELIMINAR UN USUARIO.
[ ] CREAR EN EL CONTROLADOR PARA EL PANEL ADMINISTRADOR LA FUNCION DE INTERFAZ CON LA BASE DE DATOS PARA ELIMINAR UN USUARIO Y REFLEJAR LOS CAMBIOS EN LA TABLA DE USUARIOS EN LA PAGINA BAJAUSUARIO.
[ ] CREAR UN MODAL DIALOG PARA CADA UNO DE LOS ESTADOS DE UN CAMBIO MOSTRANDO LAS OPCIONES QUE CORRESPONDAN CORRESPONDAN EN CADA CASO E IMPLEMENTAR DICHAS OPCIONES.

COSAS QUE SE PODRIAN MEJORAR:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
[ ] CADA UNO DE LOS COMBOS EN LAS VISTAS DE USUARIO SE DEBERIAN LLENAR CON LOS DATOS DE DB. POR EJEMPLO, EL COMBO DE PERFILES EN LA PAGINA ALTAUSUARIO SE DEBERIA LLENAR CON LOS PERFILES ALMACENADOS EN LA DB.
[ ] COLOCAR UN INDEX EN CADA UNO DE LOS DIRECTORIOS QUE REDIRECCIONE A LOS USUARIOS A DONDE CORRESPONDA PARA PROTEGER LOS ARCHIVOS EN CADA DIRECTORIO.
